# Nollie
Nollie Skateboards nació en 2000. El Burro y Beto, con la misión de crear tablas de skate de la mejor calidad. Nollie combina experiencia y dedicación para ofrecer productos resistentes, cómodos y funcionales, pensados para la comunidad de skaters que disfruta del street. Descubre nuestras decks y sigue la evolución del skate callejero con Nollie.
